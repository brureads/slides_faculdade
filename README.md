<h2>🌸 Sobre o repositório</h2>

<p>
  Este repositório foi criado para guardar e organizar os <strong>slides em PDF das minhas aulas da faculdade</strong>.
</p>

<p>
  A ideia é deixar tudo separado por disciplina, facilitando na hora de estudar, revisar para provas,
  consultar conteúdos antigos e manter os materiais sempre acessíveis.
</p>

<blockquote>
  Um cantinho acadêmico para deixar os estudos mais organizados e menos caóticos. 💻📖✨
</blockquote>
